\textbf{Adaptation}
\begin{itemize}
	\item 기존 국내외 진료지침이 전체의 핵심질문을 모두 포함하는 경우
	\item 관련 국내외 진료지침이 3-5년 이내에 개발되었고 결정적인 추가 근거가 없는 경우
(근거 팽창 속도가 빠른 경우 3년)
	\item 관련 국내외 진료지침(혹은 systematic review)이 근거기반 방법론*을 사용한 경우
	\item 관련 국내외 진료지침이 국가(다국가 포함) 혹은 대표적인 학회에서 개발한 경우
\end{itemize}
